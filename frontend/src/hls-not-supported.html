HLS is not supported

This website uses a video format called HLS, and uses a library called hls.js to make this work in most modern browsers. Unfortunately either you are using a browser that is not supported or have a configuration that prevents it from working.

What can you do

We test against the latest versions of Firefox, Safari and Chrome.

(We aren't checking for these, so any other browser supported by hls.js should work too.)

If you think this is an error, please contact the organisers.
